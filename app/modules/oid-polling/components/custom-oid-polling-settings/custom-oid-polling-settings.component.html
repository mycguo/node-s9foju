OID index must:

  • Only contain 0-9 and .
  • Begin with either .1 or .0
  • Have at least 2 decimal separated numbers (ex. .0.98)
  • Be valid for all selected devices

This determines how to process the differences between each polled value.

Delta: The difference between each poll value

Rate: The delta with respect to the system time difference between each poll value

 This action will be applied to the polled values after processing.
To avoid any conversion, select conversion type “Multiply” with a conversion factor of “1”. This value will be applied with the conversion type to the polled values after processing.
